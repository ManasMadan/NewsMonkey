## Run Locally

Clone the project

```bash
  git clone https://github.com/ManasMadan/NewsMonkey
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Add Api Key in a file .env.local

Start the server

```bash
  npm run start
```
